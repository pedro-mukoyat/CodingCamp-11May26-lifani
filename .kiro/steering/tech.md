# Tech Stack

## Languages & Standards
- HTML5
- CSS3
- Vanilla JavaScript (ES6+)

## Allowed External Libraries
- **Chart.js** — loaded via CDN in the `<head>` of `index.html`. No other external libraries are permitted.

## Strictly Prohibited
- Frameworks or libraries: React, Vue, Angular, jQuery, or any similar tool
- Build tools: Webpack, Vite, Parcel, npm scripts, etc.
- Any backend or server-side code
- Any database or remote storage — use **Browser Local Storage API only**

## Storage
All data persistence must use `localStorage`. No cookies, sessionStorage, IndexedDB, or remote APIs.

## Running the Project
Open `index.html` directly in a browser. No build step or local server is required.
